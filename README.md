# VideoGames
VIdeo Games Sales
The Video Games Sales dataset was analyzing by looking at the global sales, critic and user score for the two genres: shooter and sports. 
The insights found was the difference in score between the critic and user score for the two genres. This could also have impacted/influence the global sales of each genre. 
The data was first upload into SQL, from there the data was transformed using transformation functions, then aggregation functions were used to summarize the data set. Finally, visualizations were created to communicate the in a different format and to visually compare the dataset.
